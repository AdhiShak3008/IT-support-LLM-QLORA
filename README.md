# IT Support LLM using QLoRA

## Overview

Production-style domain-specific LLM fine-tuning project using QLoRA on Mistral-7B-Instruct for IT support and troubleshooting tasks.

## Features

- QLoRA 4-bit fine-tuning
- Domain-specific instruction tuning
- Evaluation and benchmarking pipeline
- Hallucination analysis
- Quantized inference
- Gradio inference UI
- Dockerized deployment workflow

## Tech Stack

- Python
- Hugging Face Transformers
- PEFT
- TRL
- bitsandbytes
- Gradio
- PyTorch
